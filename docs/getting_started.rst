Getting Started
===============

Flask-Mongo-DRF helps you build RESTful APIs with Flask and MongoDB, just like Django REST Framework.

**Core concepts:**

- **Model**: Encapsulates MongoDB Collection operations
- **Serializer**: Defines field validation and (de)serialization
- **ViewSet**: Automatically provides CRUD endpoints
- **Filter**: Converts query parameters to MongoDB filter conditions
- **Pagination**: Built-in pagination support
