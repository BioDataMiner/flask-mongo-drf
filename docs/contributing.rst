Contributing
============

We welcome contributions, bug reports, and documentation improvements.

Development Setup
-----------------

.. code-block:: bash

   git clone https://github.com/BioDataMiner/flask-mongo-drf.git
   cd flask-mongo-drf
   pip install -e .[dev]

Run Tests
---------

.. code-block:: bash

   pytest tests/

Code Style
----------

Use `black` and `flake8` to maintain consistency.
